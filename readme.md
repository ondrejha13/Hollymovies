# SDAcademy: PythonRemoteCZ23 - Hollymovies project

## Instalation and setup

```bash
pip install django
```
Create Django project:
```bash
django-admin startproject hollymovies .
```

## Project structure

- Hollymovies - settings of our project

  - settings.py
